# Support

Use GitHub Issues for bug reports, feature requests, and support questions:

- Bug reports: include the Fit File Viewer version, operating system, reproduction steps, and any relevant `.fit` file details.
- Feature requests: describe the workflow, expected behavior, and why the existing UI does not cover it.
- Security reports: do not open a public issue. Follow [SECURITY.md](SECURITY.md).
