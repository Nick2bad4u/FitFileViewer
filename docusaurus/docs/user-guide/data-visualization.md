# Data Visualization

FitFileViewer provides multiple ways to visualize your fitness data.

## Overview

Each visualization type serves a different purpose:

| Tab        | Purpose                  | Key Features              |
| ---------- | ------------------------ | ------------------------- |
| 🗺️ Map     | Geographic visualization | Route, markers, elevation |
| 📊 Charts  | Performance analysis     | Trends, comparisons       |
| 📋 Tables  | Detailed data            | Search, sort, export      |
| 📄 Summary | Quick overview           | Key metrics, stats        |

## 🗺️ Maps

See [Maps Documentation](/docs/visualization/maps) for detailed information.

### Quick Overview

- Interactive GPS route display
- Multiple map tile providers
- Lap and waypoint markers
- Measurement tools
- Fullscreen mode

## 📊 Charts

See [Charts Documentation](/docs/visualization/charts) for detailed information.

### Available Charts

| Chart      | Data Shown         | Use Case           |
| ---------- | ------------------ | ------------------ |
| Speed/Pace | Velocity over time | Analyze pacing     |
| Heart Rate | HR over time       | Monitor effort     |
| Elevation  | Altitude profile   | Understand terrain |
| Power      | Power output       | Cycling analysis   |
| Cadence    | Steps/RPM          | Technique review   |

### Interactions

- **Zoom**: Mouse wheel
- **Pan**: Click and drag
- **Reset**: Double-click
- **Hover**: View data points

## 📋 Tables

See [Tables Documentation](/docs/visualization/tables) for detailed information.

### Features

- Sortable columns
- Text search/filter
- Export to CSV
- Pagination for large datasets

## 📄 Summary

See [Summary Documentation](/docs/visualization/summary) for detailed information.

### Key Metrics

- Activity date and type
- Duration and distance
- Average and max speed
- Heart rate zones
- Calories burned
- Elevation gain/loss
