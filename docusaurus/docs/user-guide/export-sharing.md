## Exporting Tables

### Copy as CSV

1. Go to the **Tables** tab
2. Click the **Copy** button or use `Ctrl/Cmd + C`
3. Paste into Excel, Google Sheets, or text editor

### Filtered Export

1. Apply filters to show desired data
2. Export only visible rows
3. Use pagination to export sections

## Exporting Charts

### Save as Image

1. Navigate to the **Charts** tab
2. Right-click on the chart
3. Select **Save as Image**
4. Choose PNG or JPEG format
5. Select save location

### Chart Export Quality

| Format | Quality | File Size | Use Case |
|--------|---------|-----------|----------|
| PNG | High | Medium | General sharing |
| JPEG | Medium | Small | Quick sharing |

## Exporting Maps

### Map Screenshot

1. Go to the **Map** tab
2. Arrange the view as desired
3. Right-click ‚Üí **Save Map Image**
4. Choose format and save

### Tips for Map Export

- Zoom to show entire route
- Choose appropriate map layer
- Consider light/dark theme

## Printing

### Print Current View

1. Navigate to desired tab
2. **File ‚Üí Print** or `Ctrl/Cmd + P`
3. Configure printer settings
4. Print or save as PDF

### Print to PDF

Most systems support "Print to PDF":

1. Use the Print function
2. Select PDF printer
3. Choose save location

## Sharing

### Social Media

Best practices for sharing:

1. Export charts/maps as PNG
2. Crop to highlight key data
3. Add context in your post

### Fitness Platforms

FitFileViewer is for viewing/analysis. To upload to platforms:

- Keep your original FIT files
- Upload directly to platforms like Strava, Garmin Connect

## Data Privacy

### What's Stored Locally

- Recent files list
- Theme preferences
- Window size/position

### What's NOT Stored

- Your FIT file data
- GPS coordinates
- Personal information

### Privacy Tips

- FitFileViewer works offline
- No data is sent to servers
- Your files stay on your computer
